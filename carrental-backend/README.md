# Car Rental System - Backend\n\nA comprehensive Spring Boot application for managing car rentals with role-based access control, multi-city support, and complete billing system.\n\n## Features\n\n### Core Features\n- **Role-Based Access Control**: Admin and Customer roles\n- **Multi-City Support**: Chennai, Madurai, Trichy, Kanyakumari, Trivandrum\n- **Vehicle Management**: 6 categories (Mini, SUV, Luxury, Hatchback, Sedan, Electric)\n- **Booking System**: Complete booking lifecycle management\n- **Payment Processing**: Advance and rental payments with flexible options\n- **Billing System**: Automatic bill generation with extra charges calculation\n- **Notifications**: Email (Gmail) and WhatsApp (Twilio) notifications\n\n### Technical Features\n- JWT Authentication\n- CORS Configuration\n- JPA/Hibernate ORM\n- MySQL Database\n- RESTful APIs\n- Transaction Management\n- Scheduled Tasks\n\n## Technology Stack\n\n- **Framework**: Spring Boot 3.x\n- **Language**: Java 17+\n- **Database**: MySQL 8.0+\n- **ORM**: JPA/Hibernate\n- **Security**: Spring Security + JWT\n- **Build Tool**: Maven\n- **Additional Libraries**:\n  - Lombok (Boilerplate reduction)\n  - Twilio SDK (WhatsApp)\n  - Spring Mail (Email)\n  - JWT (jjwt)\n\n## Project Structure\n\n```\nsrc/main/java/com/carrental/\nв”њв”Ђв”Ђ entity/              # JPA Entities\nв”‚   в”њв”Ђв”Ђ User.java\nв”‚ API Endpoints\n\n### Authentication\n- `POST /api/auth/register/customer` - Register as customer\n- `POST /api/auth/register/admin` - Register as admin\n- `POST /api/auth/login` - Login\n\n### Vehicles\n- `GET /api/vehicles/{id}` - Get vehicle details\n- `GET /api/vehicles/city/{cityId}` - Get vehicles by city\n- `GET /api/vehicles/city/{cityId}/available` - Get available vehicles\n- `GET /api/vehicles/city/{cityId}/category/{categoryId}/available` - Get available vehicles by category\n- `POST /api/vehicles` - Create vehicle (Admin)\n\n### Bookings\n- `POST /api/bookings` - Create booking\n- `GET /api/bookings/{id}` - Get booking details\n- `GET /api/bookings/customer/{customerId}` - Get customer bookings\n- `POST /api/bookings/{id}/confirm` - Confirm booking\n- `POST /api/bookings/{id}/start` - Start rental\n- `POST /api/bookings/{id}/complete` - Complete rental\n- `POST /api/bookings/{id}/cancel` - Cancel booking\n\n### Payments\n- `POST /api/payments/advance/{bookingId}` - Process advance payment\n- `POST /api/payments/rental/{bookingId}` - Process rental payment\n- `POST /api/payments/extra-charges/{bookingId}` - Process extra charges\n- `GET /api/payments/booking/{bookingId}` - Get booking payments\n\n### Bills\n- `POST /api/bills/generate/{bookingId}` - Generate bill\n- `GET /api/bills/{id}` - Get bill details\n- `GET /api/bills/booking/{bookingId}` - Get bill by booking\n- `GET /api/bills/customer/{customerId}` - Get customer bills\n- `POST /api/bills/{id}/mark-paid` - Mark bill as paid\n- `POST /api/bills/{id}/send-notification` - Send bill notification\n\n## Configuration\n\n### Email Configuration (Gmail)\n1. Enable 2-factor authentication on Gmail\n2. Generate App Password\n3. Update `application.properties` with app password\n\n### Twilio Configuration\n1. Create Twilio account\n2. Get Account SID and Auth Token\n3. Get Twilio phone number\n4. Update `application.properties`\n\n## Database Schema\n\n### Users Table\n- Stores customer and admin information\n- Roles: CUSTOMER, ADMIN, SUPER_ADMIN\n\n### Vehicles Table\n- Vehicle details with pricing\n- Status: AVAILABLE, RENTED, MAINTENANCE\n- Associated with City and Category\n\n### Bookings Table\n- Booking lifecycle management\n- Status: PENDING, CONFIRMED, ACTIVE, COMPLETED, CANCELLED\n- Delivery type: PICKUP, HOME_DELIVERY\n\n### Payments Table\n- Payment tracking\n- Types: ADVANCE, RENTAL, EXTRA_CHARGES\n- Status: PENDING, COMPLETED, FAILED\n\n### Bills Table\n- Bill generation and tracking\n- Includes rental cost, delivery fee, extra charges\n- Status: PENDING, PAID, OVERDUE\n\n## Development\n\n### Building\n```bash\nmvn clean build\n```\n\n### Running Tests\n```bash\nmvn test\n```\n\n### Creating JAR\n```bash\nmvn clean package\njava -jar target/carrental-system-1.0.0.jar\n```\n\n## Troubleshooting\n\n### Database Connection Issues\n- Ensure MySQL is running\n- Check credentials in application.properties\n- Verify database exists\n\n### Email Not Sending\n- Check Gmail app password\n- Enable \"Less secure app access\" if needed\n- Verify SMTP settings\n\n### Twilio Issues\n- Verify Account SID and Auth Token\n- Check phone number format (+91 for India)\n- Ensure account has sufficient credits\n\n## Contributing\n\n1.
